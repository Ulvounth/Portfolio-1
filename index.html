My Projects

Community Science Museum

This was my semester project in which we were tasked with creating a website for a Community Science Museum. The intended audience comprised children between the ages of 7 and 15, as well as families with younger children and schools. The website was required to consist of a minimum of 4-6 pages, with the code written in semantically structured and neatly indented HTML. Additionally, we were expected to adhere to the DRY (Don't Repeat Yourself) principles for the CSS (Cascading Style Sheets). Furthermore, the site needed to be responsive to various screen sizes and devices.

Rainydays

This was my Cross-Course Assignment, during which we were required to create a website based on a chosen brief. In my case, the chosen brief was "Rainydays." The intended audience for this website consisted of men and women aged between 30 and 50, whose interests include outdoor activities such as hiking, exploration, skiing, camping, and canoeing.

Project Exam

This was my Project Exam, where we were tasked with creating a blog website from scratch. During this project, I independently selected the theme and content, and I was responsible for every aspect of its development, including wireframing/prototyping, design, and HTML. In addition to utilizing CSS and HTML, I employed JavaScript to retrieve data from a WordPress page containing all the blog content. As part of this project, we were required to demonstrate our proficiency in using REST APIs and other JavaScript features, such as implementing onclick modals for images and form validation for the contact form.
